== Description ==

This plugin consists in a widget which displays random lotto numbers when clicking on a button.

== Demo ==

You can play a demo of the Lotto [here](https://www.oligalma.com/downloads/javascript/lotto-numbers-generator/lotto)

== Installation ==

1. Upload the entire `lotto` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add a new Lotto widget in the sidebar you want.
